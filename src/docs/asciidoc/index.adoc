= Название сервиса
//например, TemplateApiServices - сервис шаблонов документации

*Назначение сервиса*: Сервис для описания шаблонов документации
// *Назначение сервиса*: Содержит стандатрные файлы документации на API с примерами заполнения.

<<db.adoc#, DataBase>> - описание базы данных, если используется в сервисе (описание и структура запросов в MongoDB, ES и т.д)  +
<<config.adoc#, config>> - описание конфигов (если логика метода завязана на application.yml / corp-названиеСервиса-settings)

== Методы сервиса

. <<template/template.adoc#, template>> - метод с шаблоном для описания
. <<template2.adoc#, getTemplate2>> - второй метод с шаблоном для описания
. <<template3.adoc#, getTemplate3>> - третий метод с шаблоном для описания
